Welcome to the movie app. This is the web application in which information about movies can be easily accessed.The application will be accessible for all employees within the clients network, authentication is not required.The information will be based on IMDB data provided by a publicly available OMDB API. Documentation can be found at www.omdbapi.com.The application has a menu to navigate between Search and Feature pages. The application can search movies by title.The application highlights predefined movies.
